Ayuda del puzzle

Selección de modo de juego

En el juego podrás seleccionar varios modos de juego.

  • Imágenes de la aplicación

    • Directamente al cargar el juego podras jugas con las imágenes que vienen por defecto

  • Imagenes del dispositivo

    • Selecciona 3 imagenes del dispositivo y las carga para jugar con ellas

  • Imagenes de FirebaseStorage

    • Carga las imagenes de la base de datos de Firebase y las carga para jugar con ellas

Aparte podras hacer fotos con la camara para seleccionarlas luego desde las imagenes del dispositivo

Mecanicas del juego

Cuando comienza el juego se inicia un contador que determinara lo rapido que eres haciendo los puzles

Para selecionar una pieza deja pulsado sobre la pieza que quieres mover y luego arrastrala al sitio correcto. Si no es el sitio correcto no se cambiara de sitio.

Cuando se complete el primer nivel seguira con otro puzle con más dificultad. Asi hasta 3 niveles, y se almacenara el tiempo de cada nivel para mostrarlo posteriormente en una tabla de puntuaciones.
